# 🎲 Random Task Generator — Генератор случайных задач

## Описание программы

Простое приложение с графическим интерфейсом для генерации случайных задач из предопределённых категорий (учёба, спорт, работа). Позволяет добавлять собственные задачи и типы категорий. История сгенерированных задач сохраняется и может быть отфильтрована по типу.

## Технологии

*   **Python** — язык программирования.
*   **Tkinter** — стандартная библиотека для создания GUI.
*   **random** — стандартная библиотека для генерации случайных чисел.
*   **JSON** — формат хранения данных.
*   **Git** — система контроля версий.
*   **GitHub** — удалённое хранилище кода.

## Как запустить приложение

1.  Убедитесь, что у вас установлен **Python 3.x**.
2.  Скачайте или клонируйте этот репозиторий на свой компьютер.
3.  Откройте терминал в папке проекта.
4.  Запустите приложение командой:
    ```bash
    python random_task_generator.py
    ```
5.  Интерфейс программы откроется автоматически.

## Использование программы

### Генерация задачи:
Нажмите кнопку «**Сгенерировать задачу**». В верхней части окна появится случайная задача из списка. Она автоматически добавится в историю ниже.

### Добавление новой задачи:
1.  Нажмите кнопку «**Добавить новую задачу**».
2.  В появившемся окне введите текст задачи.
3.  Затем введите тип задачи (например: «Хобби», «Дом», «Творчество»). Если такого типа нет — программа предложит его создать.
4.  Задача будет добавлена в соответствующий список и станет доступна для генерации.

### Фильтрация истории:
Используйте выпадающий список «**Фильтр по типу**», чтобы отобразить в истории только задачи определённой категории (например, только «Спорт»). Выберите «Все», чтобы показать полную историю.

### Сохранение и загрузка данных:
*   **Сохранить в JSON:** Нажмите эту кнопку для сохранения текущего списка задач и всей истории генераций в файл `tasks.json`.
*   **Загрузить из JSON:** Загружает данные из файла `tasks.json` (список задач и историю).
